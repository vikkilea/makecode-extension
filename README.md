# MakeCode Extension

This repository can be added as an **extension** in MakeCode.

## Use as Extension

1. Open [MakeCode Arcade](https://arcade.makecode.com)
2. Click on **New Project**
3. Click on **Extensions** under the gearwheel menu
4. Search for `https://github.com/vikkilea/makecode-extension` and import it

## Edit this Extension

To edit this repository in MakeCode Arcade:

1. Open [MakeCode Arcade](https://arcade.makecode.com)
2. Click on **Import** then click on **Import URL**
3. Paste `https://github.com/vikkilea/makecode-extension` and click **Import**

## Supported Targets

* for PXT/arcade
